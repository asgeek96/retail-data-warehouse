# 🏬 Retail Data Warehouse & Executive Dashboard

## 🚀 Overview

This project builds a complete end-to-end retail analytics solution using SQL Server, SSIS, and Power BI. It simulates a real-world enterprise data warehouse to track sales, customers, and profitability.

---

## 🎯 Business Problem

Retail companies face challenges such as:

* Disconnected sales and customer data
* Lack of real-time reporting
* Difficulty in tracking performance metrics

👉 This project solves it by creating a centralized data warehouse and executive dashboard.

---

## 🏗️ Architecture

Source Systems → Staging Layer → Data Warehouse → Power BI Dashboard

---

## 🗄️ Data Model

* Fact Table: Sales
* Dimension Tables: Customer, Product, Date, Region

👉 Implemented using Star Schema for optimized reporting performance.

---

## ⚙️ Tech Stack

* SQL Server (Data Warehouse)
* SSIS (ETL Pipelines)
* Power BI (Visualization)
* DAX (KPIs & Measures)

---

## 🔄 ETL Pipeline

1. Extract raw CSV data
2. Load into staging tables
3. Transform and clean data
4. Load into fact and dimension tables
5. Schedule ETL using SSIS

---

## 📊 Dashboard Features

* Sales Trends (YTD, MTD, YoY)
* Profit Analysis
* Customer Segmentation
* Regional Performance

---

## 📈 Key KPIs

* Total Sales
* Total Profit
* Profit Margin
* YoY Growth
* Customer Ranking

---

## 💡 Business Impact

* Enabled centralized reporting
* Improved decision-making speed
* Identified high-value customers and regions

---

## 🧠 Learnings
